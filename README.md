Per aixecar el docker hem d'utilitzar la comanda docker compose up -d per llavors comprovar que dins el docker desktop tenim els 4 contenidors operatius (en color verd)

/* 2 */
Per importar el nostre arxiu sql dins el phpmyadmin primer ha d'estar el docker operatiu, una vegada fet això introduim aquest ellenç dins el nostre navegador(http://localhost:8083/) i haurem d'introduir les credencials que hem assignat a l'arxiu config.js. Quan iniciem sessió tendrem una sèrie d'opcions a una barra superior i haurem de seleccionar l'opció d'importar. Una vegada fetes les passes anteriors simplement seleccionem el nostre arxiu sql y automaticamnet el phpmyadmin l¡interpretarà i ens muntarà la base de dades.
